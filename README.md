# First Vue App

Nothing big, just my first try at creating a Vue 3 app.

* CSS
* Vue.js 3 (Composition API)

## Features

* Reactive data binding with {{ }} mustache syntax 
* Vue 3 Composition API with setup() and ref() 
* Zero-config setup via CDN (no npm/build needed) 
* Clean HTML + JavaScript separation



## What I Learned

* ref() = magic variables that refresh the page automatically (reactive variables that auto-update the UI)
* createApp().mount() = connects JavaScript to HTML 
* Return data from setup() to use it in HTML
